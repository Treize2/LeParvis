Aucun lieu sélectionné

ou crée-en une nouvelle avec le bouton +.

🔄 Réimporter les horaires

Lance le scraper sur le site web déclaré et fusionne les célébrations trouvées.

🔍 Diagnostiquer (preview)

Affiche ce que le scraper voit sur le site web, sans rien écrire en base.

🔗 Fusionner avec un autre lieu

Déplace les célébrations vers une autre fiche et supprime celle-ci.

⚠️ Zone dangereuse

Supprimer définitivement ce lieu

Cette action supprime aussi toutes ses célébrations. Irréversible.

Lieux dans une zone (OpenStreetMap)

            

              Récupère tous les lieux de culte catholiques dans un rayon. Les fiches
              sont créées sans horaires ; complète ensuite avec le
              bouton « Réimporter horaires » sur chaque fiche.
            

          

        

        

          
          
        

        

          
          
        

        

          
          
        

      


      

        

          2
          

            
Depuis messes.info

            

              Renseigne une ville ou un code postal — on construit l'URL
              messes.info/horaires/<valeur> et on lance le scraper.
              Note : messes.info charge ses horaires en
              JavaScript (GWT-RPC) ; selon les pages, le SSR peut ne pas exposer
              les créneaux. Utilise « Diagnostiquer » avant pour vérifier.
            

          

        

        
        

          
          
        

      


      

        

          3
          

            
Une paroisse via son site web

            

              Récupère les horaires depuis la page d'une paroisse.
              Les sites bloquant les robots peuvent être forcés via robots.txt après
              confirmation. Pour les sites SPA (Wix, React…), active le rendu JS.
